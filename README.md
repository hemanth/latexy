# Latexy

AI-powered LaTeX editor with live preview, intelligent autocompletion, and seamless PDF export.

![Latexy Editor](screenshot.png)

## Features

- **Live Preview** - See your LaTeX render in real-time
- **AI Assistant** - Get help with LaTeX syntax and document improvements
- **Smart Autocomplete** - Context-aware LaTeX suggestions
- **PDF Export** - Download your documents instantly
- **Package Manager** - Easy LaTeX package management

## Tech Stack

- Vanilla JavaScript + Vite
- LaTeX.js for rendering
- KaTeX for math typesetting

## Development

```bash
npm install
npm run dev
```
